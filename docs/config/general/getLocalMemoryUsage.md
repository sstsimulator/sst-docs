---
title: getLocalMemoryUsage
---

Returns the current memory use as a UnitAlgebra.

## Syntax
```python
sst.getLocalMemoryUsage()
```

## Parameters
* **returns** (type: UnitAlgebra) amount of memory currently in use

## Example

```python
import sst

mem_used = sst.getLocalMemoryUsage()
```

## Import
```python
import sst
```
