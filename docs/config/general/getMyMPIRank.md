---
title: getMyMPIRank
---

Returns the MPI rank that the script is running on.

## Syntax
```python
rank = sst.getMyMPIRank()
```

## Parameters
* **returns** the rank that the script is running on

## Example

```python
import sst

rank = sst.getMyMPIRank()
```

## Import
```python
import sst
```
